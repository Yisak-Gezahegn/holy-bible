# ✝ Holy Bible — Multilingual Scripture Reader

A desktop Bible app supporting **English (KJV)**, **አማርኛ (Amharic)**, and **Afaan Oromo** — built with Electron.

## Features

- Read the full Bible in 3 languages
- Side-by-side parallel view
- Search verses across the entire Bible
- Night mode & adjustable font size
- Offline — no internet required

## Download

See the [Releases](https://github.com/Yisak-Gezahegn/holy-bible/releases) page to download the installer for Windows.

## Build from Source

```bash
git clone https://github.com/Yisak-Gezahegn/holy-bible.git
cd holy-bible
npm install
npm start          # run in development
npm run dist       # build installer
```
